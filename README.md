# GnollHack

## Introduction

GnollHack is a turn-based, traditional roguelike game built upon the codebase of the classic roguelike game [NetHack](https://github.com/NetHack/NetHack).

### Key Features

- **Modern GUI** — Animated graphics, sound effects, music, and voiceovers
- **Cross-platform** — Native support for Android, iOS, and Windows
